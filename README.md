# HiveFlow

A modern peer-to-peer file sharing system built with Rust and Next.js.

## Structure
- `/core` - Rust-based P2P networking core
- `/web` - Next.js frontend application
- `/proto` - Protocol definitions
- `/docs` - Project documentation
